# Bakeneko 65 PCB

## Features
 * Compatible with [Unified Daughterboard](https://github.com/ai03-2725/Unified-Daughterboard)
 * QMK Firmware
 * No switch LEDs and no underglow
 * All switches are south facing
 * Designed with KiCAD 5.1.6
